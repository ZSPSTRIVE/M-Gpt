## 项目简介

M-GPT 是一个面向实际使用场景打造的新一代 AI 镜像站，目标不是做一个简单的聊天页面，而是构建一个具备长期扩展能力的 AI 入口平台。

它以高完成度的对话界面为核心，同时预留了 Python API、RAG 知识库、Skills 编排、邀请码激活、工作区管理、后台管理等关键接入位，适合继续向完整产品形态演进。

可以把 M-GPT 理解为一个兼顾体验、扩展性与商业展示能力的 AI 工作台前端基础设施。

## 为什么是 M-GPT

- 对标成熟 AI 产品体验的专业级对话界面
- 支持多模型切换、按模型重答、对话管理等核心交互
- 已预留 Python / FastAPI / RAG / Skills / Admin 接口接入位
- 面向企业场景设计了账户、邀请码、工作区、能力开关等结构
- 可继续扩展为知识库平台、AI 门户、企业内部工作台或品牌镜像站

## 产品定位

M-GPT 适合以下几类场景：

- 面向个人高频用户的 AI 镜像站
- 面向团队成员统一使用的内部 AI 入口
- 面向企业客户交付的专属 AI 工作台
- 面向运营者搭建的模型聚合访问平台
- 面向知识库、流程自动化、内部助手系统的前端门户

## 核心能力

### 1. 对话工作台

当前版本已经具备完整的对话工作台骨架，包含：

- 最近对话管理
- 模型选择
- 回复复制
- 按模型重答
- 附件入口预留
- 公司知识库开关
- 分享与更多操作接口预留

### 2. 企业级扩展能力

M-GPT 并不是孤立页面，而是面向平台化能力设计的前端入口，已预留如下扩展方向：

- Python / FastAPI 服务接入
- 流式对话返回
- RAG 检索与知识源管理
- Skills 启停与编排
- 邀请码激活与登录
- 工作区状态、成员概览与后台管理

### 3. 宣传与落地兼顾

M-GPT 同时适合做产品宣传页入口和实际业务承载入口，可以用于：

- 官方镜像站发布
- 私有化 AI 门户部署
- 团队内部专属工作台
- 品牌化定制镜像站交付

## 体验方向

M-GPT 当前重点强调三件事：

- 易上手：界面足够熟悉，用户几乎不需要学习成本
- 可扩展：前端结构已经为后续接后端能力做好铺垫
- 可宣传：整体交互与产品表达更接近正式产品，而不是临时 Demo

## 对外宣传文案建议

M-GPT 是一款面向下一阶段使用需求打造的 AI 镜像站。它不仅提供更完整、更专业的对话体验，也为后续接入知识库检索、技能流转、账户体系、工作区管理与后台运营能力预留了清晰的产品结构。

如果你需要的不只是一个“能聊天”的页面，而是一个可以持续扩展、持续运营、持续对外展示的 AI 镜像站，那么 M-GPT 会是更合适的基础方案。
